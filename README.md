# 영상 악보 자동 캡처 프로그램 (Auto Score Capture)

영상 속에서 재생되는 악보를 자동으로 감지하여 캡처하고, 하나의 PDF 파일로 만들어주는 프로그램입니다.

## ✨ 주요 기능

- **GUI 제공:** 직관적인 UI를 통해 모든 기능을 쉽게 제어할 수 있습니다.
- **자동 캡처:** 악보 페이지가 넘어가는 시각적 변화를 감지하여 자동으로 캡처합니다. (OCR 방식이 아님)
- **민감도/타이머 설정:** 캡처 민감도와 시작 전 딜레이를 UI에서 직접 설정할 수 있습니다.
- **끊김 없는 PDF 생성:** 캡처된 모든 악보를 하나의 긴 이미지로 이어붙여 스크롤하기 편한 PDF로 저장합니다.

## ⚙️ 설치 및 실행 방법

### 사전 준비

이 프로그램을 사용하려면 컴퓨터에 **Python**이 설치되어 있어야 합니다.
- [python.org](https://www.python.org/)에서 최신 버전을 다운로드하여 설치하세요.
- **중요:** 설치 과정에서 **"Add Python to PATH"** 체크박스를 반드시 선택해주세요.

### 설치 과정

1.  **프로젝트 다운로드 (Clone):**
    터미널(cmd 또는 PowerShell)을 열고 아래 명령어를 입력하여 프로젝트 파일을 다운로드합니다.
    ```bash
    git clone https://github.com/YourUsername/score-capture-tool.git
    ```

2.  **폴더 이동:**
    ```bash
    cd score-capture-tool
    ```

3.  **필수 라이브러리 설치:**
    아래 명령어를 실행하면 `requirements.txt` 파일에 명시된 모든 라이브러리가 자동으로 설치됩니다.
    ```bash
    pip install -r requirements.txt
    ```

### 프로그램 실행

설치가 완료되면, 아래 명령어로 프로그램을 실행할 수 있습니다.
```bash
python score_capture_gui.py
```

## 📖 사용법

1.  프로그램이 실행되면 작은 UI 창이 나타납니다.
2.  **[1. 캡처 영역 선택]** 버튼을 누르면 화면이 캡처되고, 마우스로 악보가 표시될 전체 영역을 드래그하여 지정합니다.
3.  영역 선택이 완료되면 **[2. 캡처 시작]** 버튼이 활성화됩니다.
4.  필요에 따라 `민감도`와 `시작 딜레이` 값을 조절합니다.
5.  **[2. 캡처 시작]** 버튼을 누르면 설정된 딜레이 시간만큼 카운트다운 후 자동 캡처를 시작합니다.
6.  악보 캡처를 마치려면 **[종료 및 PDF 생성]** 버튼을 누릅니다.
7.  프로그램 폴더에 `final_sheet_music_stitched.pdf` 파일이 생성됩니다.
